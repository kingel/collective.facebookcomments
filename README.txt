Introduction
============

Add Facebook Comments to your pages. 

This packages insert two viewlets into your pages to be able to show the commentsbox. 

You need to configure your Facebook AppID in the control panel configlet

(You can get it from https://developers.facebook.com/apps )

In this configlet you can also configure for which portal_types to show the Facebook Comments
